API Basejump: URL Shortener

User Stories

I can pass a URL as a parameter and I will receive a shortened URL. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead. When I visit that shortened URL, it will redirect me to my original link.

Example usage:

https://url-shortener-express-gomez-angel.c9users.io/new/http://www.ebay.com
https://url-shortener-express-gomez-angel.c9users.io/new/http://www.google.com
https://url-shortener-express-gomez-angel.c9users.io/new/http://www.amazon.com
https://url-shortener-express-gomez-angel.c9users.io/new/http://www.google.com
hhttps://url-shortener-express-gomez-angel.c9users.io/new/http://ebay.com
https://url-shortener-express-gomez-angel.c9users.io/1

Example output:

original_url: "http://google.com", short_url: "https://url-shortener-gomez-angel.c9users.io/9"
